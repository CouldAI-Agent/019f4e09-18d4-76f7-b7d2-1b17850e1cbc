# Mecanismos Contables de Crédito

Una aplicación móvil y web construida con Flutter que describe los principales mecanismos contables de crédito utilizados en el entorno empresarial y financiero.

## Características

- Lista interactiva de mecanismos de crédito.
- Descripciones claras de opciones como Crédito Comercial, Crédito Bancario, Arrendamiento Financiero (Leasing) y Factoraje.
- Interfaz de usuario limpia y adaptable.

## Cómo ejecutar

1. Asegúrate de tener Flutter instalado.
2. Clona el repositorio.
3. Ejecuta `flutter run` en la raíz del proyecto.
